# IT Professional Learning Platform

This project is an **interactive learning and progress tracking platform** designed for individuals who want to improve their skills in the IT field.
Users can mark completed topics, track their progress, access learning resources, and test their knowledge with quizzes.

---

##  Features

*  **Progress Tracking**

  * Completed / remaining topics
  * Overall progress bar

*  **Modular Learning Structure**

  * Networking Fundamentals
  * Server & Infrastructure
  * Virtualization
  * Security & Firewall
  * Business Analysis & Software

*  **Category Filtering**

  * View topics based on categories

*  **Checklist System**

  * Mark completed topics
  * Automatic progress calculation

   **Quiz System**

  * Topic-based questions
  * Instant correct / incorrect feedback
  * Answer explanations

*  **Note-Taking Area**

  * Add personal notes for each topic

*  **Resource Management**

  * Topic-specific documentation links

*  **Priority Roadmap**

  * Learning path guidance
  * Focus on critical topics
